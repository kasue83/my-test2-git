# Fullstack Board Application

## Project Setup Instructions
1. Clone the repository: `git clone https://github.com/kasue83/my-test2-git.git`
2. Navigate to the project directory: `cd my-test2-git`
3. Install dependencies:
   - For the backend: `npm install` (inside the server directory)
   - For the frontend: `npm install` (inside the client directory)
4. Run the server: `npm start` (inside the server directory)
5. Run the frontend: `npm start` (inside the client directory)

## Features
- User authentication
- Board creation and management
- Drag-and-drop functionality
- Real-time updates
- Responsive design

## Technology Stack
- Frontend: React, Redux
- Backend: Node.js, Express
- Database: MongoDB
- Real-time communication: Socket.io
